# Time Calculator

Веб-приложение для расчёта временных метрик по UTC+0.

## Что делает приложение
- Рассчитывает часы от начала года (21 марта 00:00), месяца, недели.
- Вычисляет пропорциональные значения для годовых, месячных и недельных периодов.
- Минималистичный веб-интерфейс в стиле Wall Street.

## Формат ввода
Введите дату и время в формате: `DD.MM HH`  
Пример: `05.05 05` (5 мая, 5 часов)

## Расчёты
- **Год**: часы / 8760 * (120, 168, 24, дни_месяца)
- **Месяц**: часы / дни_месяца*24 * (120, 168, 24)
- **Неделя**: часы / 120 * 24 и часы / 168 * 24
